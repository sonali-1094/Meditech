# Dataset Structure

This folder contains small starter datasets so the ML training scripts run immediately.

- `disease_symptoms.csv`: symptom text mapped to broad disease or care categories.
- `mental_health_sentiment.csv`: mood journal text mapped to `positive`, `neutral`, `negative`, or `crisis`.
- `health_risk.csv`: tabular health metrics mapped to `low`, `medium`, or `high` risk.

For production, replace these starter rows with clinically reviewed, consented, privacy-safe datasets. Keep personally identifiable information out of training data.
